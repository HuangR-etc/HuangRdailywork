# 概览

## 单细胞质控
单细胞分析流程中的细胞质量控制步骤，基于 nFeature_RNA、nCount_RNA 和线粒体比例进行细胞过滤。

这是一个基于 Adam Tool Protocol (ATP) 开发的工具，提供了完整的单细胞 QC 功能实现和用户界面。该工具读取上游标准化 kit 输出的 Seurat 对象，执行质量控制，生成过滤后的对象和配套 QC 图，作为后续预处理/降维/聚类流程的直接输入。

## 主要功能
- **读取标准化对象**: 读取 standardized_object.rds（Seurat 对象）
- **计算 QC 指标**: 自动计算 nFeature_RNA、nCount_RNA 和线粒体比例
- **手工阈值过滤**: 基于用户定义的阈值过滤细胞
- **生成 QC 图**: 生成过滤前后的 violin plot 和 scatter plot
- **输出过滤对象**: 生成 qc_object.rds 供下游流程使用
- **完整报告**: 生成详细的处理报告和统计摘要

# 特性

## 核心特性

### 易用性
- 直观的表单界面配置所有 QC 参数
- 详细的参数说明和默认值提示
- 自动检测输入文件并验证对象类型
- 完整的错误处理和用户友好的错误消息

### 可靠性
- 基于 Seurat 官方 QC 流程的稳定实现
- 完整的输入验证和错误检查
- 详细的日志记录和调试信息
- 支持多种图像格式输出（PDF/PNG）

### 扩展性
- 模块化的代码结构，便于添加新的 QC 方法
- 预留 scater_mad 和 per_sample_manual 方法接口
- 灵活的阈值配置，支持启用/禁用各个阈值
- 可配置的输出对象类型（Seurat/继承输入类型）

## 技术特性

### 输入支持
- 支持 Seurat 对象输入（当前版本）
- 自动检测和计算缺失的 QC 指标
- 支持自定义线粒体基因匹配模式
- 验证 assay 存在性和可用性

### 过滤逻辑
- 手工阈值过滤方法（manual）
- 支持多阈值组合过滤
- 详细的删除原因记录
- 防止过滤后无细胞剩余的检查

### 输出文件
- qc_object.rds: 过滤后的 Seurat 对象
- qc_metrics_all_cells.csv.gz: 所有细胞的 QC 指标
- qc_kept_cells.csv.gz: 保留细胞的 QC 指标
- qc_removed_cells.csv.gz: 删除细胞的 QC 指标
- qc_summary.json: 统计摘要 JSON
- report.md: 详细处理报告
- plots/: QC 图像目录（PDF/PNG 格式）

## 默认阈值说明

工具提供一组合理的默认阈值，基于 Seurat 官方 PBMC3K 教程：
- **最小基因数**: 200（nFeature_RNA > 200）
- **最大基因数**: 2500（nFeature_RNA < 2500）
- **最大线粒体比例**: 5%（percent.mt < 5）
- **最小/最大 UMI 数**: 0（默认不启用）

这些默认值可作为起始模板，用户应根据具体实验数据调整。

## 工作流程

1. **读取输入**: 读取 standardized_object.rds（Seurat 对象）
2. **计算指标**: 计算或验证 nFeature_RNA、nCount_RNA、percent.mt
3. **生成前图**: 生成过滤前的 violin plot 和 scatter plot
4. **执行过滤**: 根据用户阈值过滤细胞
5. **生成后图**: 生成过滤后的 violin plot 和 scatter plot
6. **保存输出**: 保存过滤后的对象和所有 sidecar 文件
7. **生成报告**: 创建详细的处理报告和统计摘要

## 错误处理

工具包含完整的错误处理机制：
- 输入文件不存在或损坏
- 对象类型不支持（非 Seurat 对象）
- assay 不存在或不可用
- 过滤后无细胞剩余
- R 环境或包缺失
- 图像生成失败

所有错误都会提供清晰的错误消息和解决建议。
